Dom.js ผมฝึกการใช้Dom ตามที่อาจารย์สอนขยายความเข้าใจขึ้น
โดยผมให้ Ai สร้างโจทย์ขึ้นมาให้เสริมความเข้าใจ
โดยผมจดโค้ดที่ผมไม่รู้จากการดูวิดีโอย้อนหลัง (ไม่ได้มาเรียน)
เช่น children ต่างจาก childNode อย่างไร ทำงานอย่างไร
การ create Element จะสร้างอยู่ที่ document  ที่เป็น root node ก่อน แล้ว มันจะไปเชื่อมกับส่วนอื่นๆใน Tree ได้โดยใช้ appendChild บอกว่าElement ใหม่ต้อง
อยู่ตรงนี้นะ เป็น ListNode

querySelector มันใช้หาตัวแรกทีเจอ ตรงกันเช่น h1#FirstName ถ้ามันตรงทั้ง Element และ ID มันจะดึง Element ตัวแรกออกมาให้ ถ้าไม่มีจะได้ null

ซึ่งตอนทำก็ได้เข้าใจมากขึ้นว่าต้องใช้อะไร โดยที่ตอนทำผมใช้สไลด์อาจารย์ประกอบ
สิ่งที่ได้รู้คือ ผมรู้วิธีสร้าง และ ต่อ Elementใหม่ childNode มันจะใช้ For ได้ มันจะเอา whitespace ออกมากด้วย ส่วน children มันจะให้ Element ทุกตัวออกมาเลย ถ้าอยากได้ข้อมูลต้องใช้ Array.From
ส่วน getElementBy Classname , ID , TAG ก็ทำงานคล้ายกันแค่ระบุชื่อ พวกนั้นให้ตรงกันก็พอ

Twosum.js
เป็นการทบทวนความรู้ เรื่องเดิมๆเพื่อให้จำได้ดียิ่งขึ้น
โจทย์นี้ คือ ให้หา ตำแหน่ง array ที่ บวกกันแล้วได้ target ที่กำหนด และ ต้องเป็นตำแหน่งล่าสุดที่หาเจอเท่านั้น
ปัญหาที่เจอคือ [2,5,5,11] มัน + ตัวมันเองทำให้ต้องใช้ if เพิ่ม
หรือ บางทีเราเจอคำตอบแล้ว แต่โจทย์หาตำแหน่งล่าสุด
